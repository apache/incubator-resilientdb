32// Key Value Storage supporting two types of interfaces:
33// Non-version:
34// It provides set and get function to set a value to a specific key
35// Values will be set directly.
36// Version:
37// It provides set and get function to set a value to a specific key
38// with a version.
39// The version inside setting function is to support OCC verification.
40// The version value should be obtained before setting
41// and returned back when setting a new value. If the version is not
42// the same as the old one, return failure.
43// If the value of a specific version does not exist or providing
44// version 0 as parameter when accessing get
45// it returns the current value along its version.
46//
47// Note: Only one type of interface are allowed to be used.
48//
49
